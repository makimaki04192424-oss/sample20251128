# sample20251128

## このWEBページについて

「みんなでれんしゅう」は、みんなで集まって練習や復習ができるシンプルな静的サイトです。

### 機能

- **ホーム**: おへやをつくったり、おともだちとさんかしたりできます
- **れんしゅう**: コーディングチャレンジ、スピード反復、クイズなどの練習ができます
- **せつめい**: サイトの目的と拡張案について説明しています

### 拡張案

- WebSocketでリアルタイム通信
- ユーザー認証
- 課題・進捗トラッキング
